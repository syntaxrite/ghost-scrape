Quick Start

Drop this into Node, Python, or cURL.

cURL
curl -X POST https://YOUR_DOMAIN/api/scrape \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_KEY" \
  -d '{"url":"https://example.com"}'
Node.js
const res = await fetch("https://YOUR_DOMAIN/api/scrape", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": process.env.GHOST_API_KEY
  },
  body: JSON.stringify({ url: "https://example.com" })
});

const data = await res.json();
console.log(data.markdown);
Python
import os
import requests

res = requests.post(
    "https://YOUR_DOMAIN/api/scrape",
    headers={"x-api-key": os.getenv("GHOST_API_KEY")},
    json={"url": "https://example.com"}
)

print(res.json()["markdown"])

What counts as one request: one page sent through Ghost Scrape and returned as clean markdown.
