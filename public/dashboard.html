Quick Start

Drop this into Node, Python, or cURL.

cURL
curl -X POST https://YOUR_DOMAIN/api/scrape \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_KEY" \
  -d '{"url":"https://example.com"}'
Node.js
fetch("/api/scrape", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": process.env.GHOST_API_KEY
  },
  body: JSON.stringify({ url: "https://example.com" })
})
Python
import requests

requests.post(
    "https://YOUR_DOMAIN/api/scrape",
    headers={"x-api-key": "YOUR_KEY"},
    json={"url": "https://example.com"}
)

What counts as one request: one page sent through Ghost Scrape and returned as clean markdown.
