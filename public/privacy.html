Privacy Policy

Last updated: April 2026

This Privacy Policy explains how Ghost Scrape collects, uses, and protects your data when you use our API and website.

1. Data We Collect

  • Email address (for authentication)
  • API usage logs (requests, timestamps)
  • IP address (for abuse prevention)
  • URLs submitted to the API (temporary processing)

2. How We Use Data

  • Authenticate users and API keys
  • Enforce rate limits and prevent abuse
  • Improve API performance and reliability
  • Monitor system security

3. Data Storage

We store data securely using trusted third-party providers such as Supabase and Vercel. Data is retained only as long as needed to operate the service.

4. Data Sharing

We do not sell, rent, or trade your data. We only share it with essential infrastructure providers required to run the service.

5. API Usage

You are responsible for all requests made using your API key. We log usage for rate limiting and abuse prevention.

6. Security

We use reasonable security measures, but no system is completely secure. Keep your API key private at all times.

7. Your Rights

  • Request account deletion
  • Revoke API access
  • Request data information
