Ghost Scrape API

Convert web pages into clean, structured markdown for LLMs, agents, and RAG pipelines.

Auth

x-api-key

Input

POST JSON

Output

Markdown

Request

POST /api/scrape
Content-Type: application/json
x-api-key: ghost_xxx

{
  "url": "https://example.com"
}

cURL Example

curl -X POST "https://ghost-scrape.tech/api/scrape" \
  -H "Content-Type: application/json" \
  -H "x-api-key: ghost_xxx" \
  -d '{"url":"https://example.com"}'

Response

{
  "success": true,
  "title": "Example Title",
  "domain": "example.com",
  "source": "axios",
  "wasBlocked": false,
  "wordCount": 1200,
  "markdown": "# Clean content..."
}

JavaScript

const res = await fetch("https://ghost-scrape.tech/api/scrape", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "ghost_xxx"
  },
  body: JSON.stringify({
    url: "https://example.com"
  })
});

const data = await res.json();
console.log(data.markdown);

Errors

401

Invalid API key

429

Rate limited

422

Extraction failed
