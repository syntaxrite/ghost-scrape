API docs

Clean Markdown for LLMs and RAG.

Send a public URL to /api/scrape with your API key and get back structured Markdown plus metadata.

Auth
x-api-key
Method
POST
Free tier
300 / month

Request

POST /api/scrape
Content-Type: application/json
x-api-key: ghost_xxx

{ "url": "https://example.com/article" }

Response

{
  "success": true,
  "title": "Example Article",
  "domain": "example.com",
  "canonicalUrl": "https://example.com/article",
  "source": "axios",
  "sourceType": "generic",
  "wasBlocked": false,
  "wordCount": 1200,
  "excerpt": "Short summary...",
  "headings": ["Intro", "Main Topic"],
  "author": "Author Name",
  "publishedAt": "2024-01-01",
  "duration_ms": 842,
  "markdown": "# Clean content here..."
}

JavaScript

const res = await fetch("https://ghost-scrape.tech/api/scrape", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "ghost_xxx"
  },
  body: JSON.stringify({ url: "https://example.com/article" })
});

const data = await res.json();
console.log(data.markdown);

Limits

Authenticated users receive a free monthly quota of 300 requests.

There is no separate daily cap in the current beta.

Every request from the dashboard playground or your own app counts against the same monthly quota.

401

Missing API key.

403

Invalid API key.

429

Monthly quota reached or request burst blocked.
